# S3 extension

How to add S3-compatible object storage.

## Dependencies

Install the AWS SDK in the package or app that owns object storage access:

```bash
pnpm --filter=<backend-or-shared-package> add @aws-sdk/client-s3
```

Install the presigner only if browsers need direct upload/download URLs:

```bash
pnpm --filter=<backend-or-shared-package> add @aws-sdk/s3-request-presigner
```

## Local development

Use a dedicated env file for the LocalStack container so app credentials are not passed into the LocalStack process.

Create `.env.localstack`:

```bash
SERVICES=s3
DEBUG=0
DATA_DIR=/data
LS_LOG=warn
AWS_DEFAULT_REGION=us-east-1
```

Merge this into `zap.yaml`:

```yaml
ports: [LOCALSTACK_PORT]
docker:
  localstack:
    image: localstack/localstack:latest
    ports:
      - "${LOCALSTACK_PORT}:4566"
    env:
      - .env.localstack
    volumes:
      - localstack-data:/data
```

Add app-local object storage variables:

```bash
AWS_REGION=us-east-1
AWS_ACCESS_KEY_ID=test
AWS_SECRET_ACCESS_KEY=test
AWS_ENDPOINT_URL=http://localhost:${LOCALSTACK_PORT}
AWS_S3_BUCKET=<local-bucket-name>
```

The port will be assigned automatically by zapper.

### Local troubleshooting

- Use lowercase bucket names without underscores.
- Set `forcePathStyle: true` whenever `AWS_ENDPOINT_URL` is set.
- If browser uploads fail, check LocalStack bucket CORS.
- Presigned LocalStack URLs point at localhost, so external services cannot fetch them.

## Code

Use LocalStack locally and the AWS SDK in application code. The app switches to LocalStack when `AWS_ENDPOINT_URL` is set.

### S3 client

```typescript
import { S3Client } from "@aws-sdk/client-s3";

export interface S3Config {
  AWS_REGION: string;
  AWS_ACCESS_KEY_ID: string;
  AWS_SECRET_ACCESS_KEY: string;
  AWS_ENDPOINT_URL?: string;
}

let s3Client: S3Client | null = null;

export const getS3Client = (env: S3Config): S3Client => {
  if (s3Client) return s3Client;

  const config: ConstructorParameters<typeof S3Client>[0] = {
    region: env.AWS_REGION,
    credentials: {
      accessKeyId: env.AWS_ACCESS_KEY_ID,
      secretAccessKey: env.AWS_SECRET_ACCESS_KEY,
    },
  };

  if (env.AWS_ENDPOINT_URL) {
    config.endpoint = env.AWS_ENDPOINT_URL;
    config.forcePathStyle = true;
  }

  s3Client = new S3Client(config);
  return s3Client;
};
```

`forcePathStyle: true` is needed for LocalStack because it uses path-style bucket URLs such as `localhost:4566/bucket/key`.

### Local bucket bootstrap

In local development, buckets can be created automatically before upload operations:

```typescript
import {
  CreateBucketCommand,
  HeadBucketCommand,
  type S3Client,
} from "@aws-sdk/client-s3";

export const ensureLocalBucketExists = async (
  client: S3Client,
  bucket: string,
  endpointUrl?: string,
): Promise<void> => {
  if (!endpointUrl) return;

  try {
    await client.send(new HeadBucketCommand({ Bucket: bucket }));
  } catch (error: unknown) {
    const err = error as {
      name?: string;
      $metadata?: { httpStatusCode?: number };
    };

    if (err.name === "NotFound" || err.$metadata?.httpStatusCode === 404) {
      await client.send(new CreateBucketCommand({ Bucket: bucket }));
      return;
    }

    throw error;
  }
};
```

If the browser uploads directly to S3 with presigned URLs, configure bucket CORS in local bootstrap too. The browser enforces CORS for LocalStack URLs.

### Upload and download

```typescript
import { GetObjectCommand, PutObjectCommand } from "@aws-sdk/client-s3";

export const uploadToS3 = async (input: {
  client: ReturnType<typeof getS3Client>;
  bucket: string;
  key: string;
  body: Buffer;
  contentType: string;
}): Promise<void> => {
  await input.client.send(
    new PutObjectCommand({
      Bucket: input.bucket,
      Key: input.key,
      Body: input.body,
      ContentType: input.contentType,
    }),
  );
};

export const downloadFromS3 = async (input: {
  client: ReturnType<typeof getS3Client>;
  bucket: string;
  key: string;
}): Promise<Buffer> => {
  const response = await input.client.send(
    new GetObjectCommand({
      Bucket: input.bucket,
      Key: input.key,
    }),
  );

  const stream = response.Body;
  const chunks: Uint8Array[] = [];
  for await (const chunk of stream as AsyncIterable<Uint8Array>) {
    chunks.push(chunk);
  }
  return Buffer.concat(chunks);
};
```

### Presigned URLs

```typescript
import { GetObjectCommand, PutObjectCommand } from "@aws-sdk/client-s3";
import { getSignedUrl } from "@aws-sdk/s3-request-presigner";

export const createPresignedS3PutUrl = async (input: {
  client: ReturnType<typeof getS3Client>;
  bucket: string;
  key: string;
  contentType: string;
}) => {
  return getSignedUrl(
    input.client,
    new PutObjectCommand({
      Bucket: input.bucket,
      Key: input.key,
      ContentType: input.contentType,
    }),
    { expiresIn: 60 * 5 },
  );
};

export const createPresignedS3GetUrl = async (input: {
  client: ReturnType<typeof getS3Client>;
  bucket: string;
  key: string;
}) => {
  return getSignedUrl(
    input.client,
    new GetObjectCommand({
      Bucket: input.bucket,
      Key: input.key,
    }),
    { expiresIn: 60 * 5 },
  );
};
```
