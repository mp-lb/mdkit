# How to update [AGENTS.md](http://AGENTS.md)

1. Make sure AGENTS.md is not dirty in working git repo
2. Clone the latest template with `dx read --store felixsebastian/mp-lb-dev agents-md-template.txt > AGENTS_NEW.md`
3. Extract project specific info: name, description, clearly project specific instructions and docs references
4. Update AGENTS_NEW.md to use it
5. Delete AGENTS.md and rename AGENTS_NEW.md to AGENTS.md
